# NewCool Conservador - Bienes Raices Chile

Portal educativo sobre Conservadores de Bienes Raices en Chile.

## Demo

[https://conservador.newcool.io](https://conservador.newcool.io)

## Caracteristicas

- Calculadora de aranceles CBR
- Buscador de conservadores por region
- Guia de inscripcion de propiedad
- Informacion sobre certificado de dominio
- Tramites inmobiliarios

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
